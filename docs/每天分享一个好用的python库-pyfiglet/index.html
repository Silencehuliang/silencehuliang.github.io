前言

学习Linux命令的朋友们肯定都学过或者听说过figlet这个有趣的命令，今天分享是Python中实现这个命令的第三方库：pyfiglet。

pyfiglet

简介

pyfiglet是用Python写的figlet命令的第三方库，可以使用字符组成ASCII艺术图片。

安装

1

pip install pyfiglet
